# Phase 48 -- UI Design Contract

> Visual and interaction contract for the project scaffold phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> **Scope note:** This phase establishes design tokens and placeholder pages only. No real UI components are built (that is Phase 50). The contract here defines the token foundation that all downstream phases consume.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | none (fully custom retro components in Phase 50) |
| Icon library | none for scaffold (deferred to Phase 50) |
| Font (body) | `system-ui, -apple-system, sans-serif` |
| Font (mono) | `'IBM Plex Mono', 'JetBrains Mono', ui-monospace, monospace` (loaded via Google Fonts CDN) |

**Rationale:** shadcn/ui is explicitly out of scope per CONTEXT.md D-01 -- it conflicts with the BAM retro industrial aesthetic. All components are custom-built in Phase 50.

---

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, input internal padding |
| md | 16px | Default element spacing, panel padding |
| lg | 24px | Section padding inside panels |
| xl | 32px | Layout gaps between panels |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level vertical spacing |

Exceptions: none

---

## Typography

| Role | Size | Weight | Line Height | Transform |
|------|------|--------|-------------|-----------|
| Body | 16px | 400 (regular) | 1.5 | none |
| Label / Button | 14px | 700 (bold) | 1.2 | uppercase |
| Heading | 20px | 700 (bold) | 1.2 | uppercase |
| Display | 28px | 700 (bold) | 1.1 | uppercase |

**Font rules (from CONTEXT.md D-05, D-06):**
- Body text: system sans-serif stack
- Data fields, codes, technical values: monospace font (IBM Plex Mono)
- Headings, tab labels, button text: `text-transform: uppercase` per BAM reference style
- Two weights only: 400 (regular) for body, 700 (bold) for headings/labels/buttons

## Border and Shadow Tokens

These are critical to the BAM retro aesthetic and must be defined in the `@theme` block.

| Token | Value | Usage |
|-------|-------|-------|
| `retro-thick` (border-width) | 3px | Standard panel/card/input outlines |
| `retro-extra-thick` (border-width) | 4px | Emphasized panels, dialog frames |
| `retro-raised` (shadow) | `2px 2px 0px #1A1A1A` | Default button state, elevated panels |
| `retro-pressed` (shadow) | `inset 2px 2px 0px rgba(0,0,0,0.3)` | Pressed/active button state |

### Hazard Stripe Utility

A reusable CSS utility class for the signature diagonal hazard stripe pattern:

```css
.bg-hazard-stripe {
  background: repeating-linear-gradient(
    -45deg,
    #FFD700,
    #FFD700 10px,
    #1A1A1A 10px,
    #1A1A1A 20px
  );
}
```

Used for: decorative panel headers, section dividers, warning borders.

## Placeholder Page Visual Contract

Since Phase 48 includes two routable placeholder pages (per success criteria), they must render with retro tokens to prove the design token pipeline works.

### Placeholder Page Layout

```
+--[ CHARCOAL BACKGROUND ]-------------------------------+
|                                                         |
|   +--[ CREAM PANEL, 3px ink border ]----------------+  |
|   |                                                   |  |
|   |  [HAZARD STRIPE BAR - 8px tall]                  |  |
|   |                                                   |  |
|   |  HEADING (20px, bold, uppercase, ink color)       |  |
|   |                                                   |  |
|   |  Body text (16px, regular, ink color)             |  |
|   |                                                   |  |
|   |  [Optional: locale switcher select]               |  |
|   |                                                   |  |
|   +--------------------------------------------------+  |
|                                                         |
+---------------------------------------------------------+
```

- Background: `retro-charcoal` (`#2A2A2A`)
- Panel: `retro-cream` (`#F5F0E1`) with `retro-thick` (3px) `retro-ink` border
- Hazard stripe bar at top of panel: 8px tall, `bg-hazard-stripe` utility
- All text in `retro-ink` (`#1A1A1A`)
- Panel centered horizontally, 24px padding inside
- Max width: 640px
- Panel shadow: `retro-raised` (2px 2px 0px #1A1A1A)

      -45deg,
