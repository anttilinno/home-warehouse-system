**Create `frontend/features/auth/components/oauth-error-handler.tsx`:**

A client component ("use client") that:
1. Uses `useSearchParams()` from `next/navigation` to read `oauth_error` query parameter
2. Uses `useTranslations("auth")` for translated messages
3. Defines an `OAUTH_ERROR_KEYS` mapping object:
   - `authorization_cancelled` -> `"oauth.errors.cancelled"`
   - `email_not_verified` -> `"oauth.errors.emailNotVerified"`
   - `invalid_state` -> `"oauth.errors.invalidState"`
   - `server_error` -> `"oauth.errors.serverError"`
   - `provider_unavailable` -> `"oauth.errors.providerUnavailable"`
4. In a `useEffect` (dependency: searchParams, t):
   - Reads `oauth_error` param
   - If it exists and maps to a known key, shows `toast.error()` from sonner with:
     - Title: `t("oauth.errors.title")`
     - Description: `t(OAUTH_ERROR_KEYS[errorCode])`
     - Duration: 8000 (longer than default so user has time to read)
   - Cleans up the URL by removing `oauth_error` param using `window.history.replaceState` (avoids full navigation)
5. Returns `null` (render-less component, side-effect only)

Wrap the component body in a `Suspense` boundary pattern: export a wrapper that wraps the actual implementation in `<Suspense>` since `useSearchParams()` requires Suspense in Next.js App Router. Alternatively, create the component as `OAuthErrorHandlerInner` and export a wrapper `OAuthErrorHandler` that wraps it in `<Suspense fallback={null}>`.

**Update `frontend/app/[locale]/(auth)/login/page.tsx`:**

1. Import `OAuthErrorHandler` from `@/features/auth/components/oauth-error-handler`
2. Add `<OAuthErrorHandler />` inside the `<CardContent>` block, before the `<SocialLogin />` component. It renders nothing visible but needs to be within the page to trigger the toast.
3. The login page is a server component -- the OAuthErrorHandler is a client component imported into it, which is the standard Next.js pattern.

Do NOT use the `useRouter` for URL cleanup -- use `window.history.replaceState` to avoid triggering a re-render/navigation cycle.
