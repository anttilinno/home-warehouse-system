## Tech Debt (Non-Blocking)

### Phase 41: Frontend OAuth Flow
- No `linkSuccess` toast after account linking — `connectedAccounts.linkSuccess` translation key exists but ConnectedAccounts component doesn't display success feedback after the link redirect chain
- No `noAccounts` empty-state message — `connectedAccounts.noAccounts` translation key exists but component renders empty list silently

### Phase 42: Error Handling & i18n
- `OAuthErrorHandler` mounted only on login page, not register page — errors from register-initiated OAuth redirect to login page (flow completes, UX is suboptimal)
- 3 orphaned translation keys defined but never referenced: `setPassword.title`, `connectedAccounts.linkSuccess`, `connectedAccounts.noAccounts`
- Backend OAuth redirect to `/auth/callback` without locale segment — non-English users see callback page in English locale (flow completes, locale context lost)

### Total: 5 items across 2 phases (all UX polish, no functional impact)
