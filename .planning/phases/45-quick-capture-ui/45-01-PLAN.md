A multi-photo thumbnail strip supporting 1-5 photos with add/remove. This component manages photo display only -- the parent component handles capture logic.

```typescript
"use client";

import { useRef, useEffect } from "react";
import { X, Camera } from "lucide-react";
import { useTranslations } from "next-intl";

interface CapturedPhoto {
  id: string;
  preview: string; // Object URL
}

interface CapturePhotoStripProps {
  photos: CapturedPhoto[];
  maxPhotos?: number;
  onTakePhoto: () => void;
  onRemovePhoto: (id: string) => void;
}
```

Implementation:
- Render a horizontal scroll container (`flex gap-2 overflow-x-auto`)
- Each photo as a 64x64 rounded thumbnail (`w-16 h-16 rounded-lg object-cover`) with an absolute-positioned X button (top-right corner) for removal. The X button should be a small circle (16x16) with bg-destructive text. Call `onRemovePhoto(photo.id)` on click.
- If `photos.length < maxPhotos` (default 5), show an "add photo" button at the end: a 64x64 dashed border box with Camera icon and the count text (e.g., "2/5"). Use `useTranslations("quickCapture")` for `t("photosCount", { count: photos.length })`. Call `onTakePhoto()` on click.
- If `photos.length >= maxPhotos`, do NOT show the add button.
- If `photos.length === 0`, show a larger take-photo button: full width, 48px tall, with Camera icon and `t("takePhoto")` text. Call `onTakePhoto()` on click.
- Minimum touch target of 44px for all interactive elements.
