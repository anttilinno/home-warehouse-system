- Sessions: `GET /users/me/sessions` -> SessionResponse[] `{ id, device_info(parsed UA label), ip_address?, last_active_at, created_at, is_current }`. `DELETE /users/me/sessions/{id}` (400 if id===current). `DELETE /users/me/sessions` (revoke all-others; 400 if no current session — now WORKS via Plan 01 CurrentSession middleware).

OAuth-only (has_password=false) accounts: UpdatePassword ALREADY accepts empty/omitted current_password (RESOLVED Q1 — service.go:172) -> set-password path sends only new_password.
- Identity: `GET /users/me` -> includes `has_password` (drives change-vs-set password + unlink lockout).
- Can-delete: `GET /users/me/can-delete` -> `{ can_delete, blocking_workspaces:[{id,name,slug}] }`.
- Delete: `DELETE /users/me { confirmation: "DELETE" }` (case-insensitive; 409 if sole owner) -> clears cookies.
