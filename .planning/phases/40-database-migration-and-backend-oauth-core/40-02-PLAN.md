<task type="auto">
  <name>Task 2: Create OAuth provider configs, profile fetchers, and FindOrCreateUser service</name>
  <files>

  <action>
    **Providers (providers.go):**
    1. Create `GoogleConfig(clientID, clientSecret, backendURL string) *oauth2.Config`:
       - RedirectURL: `backendURL + "/auth/oauth/google/callback"`
       - Scopes: `[]string{"openid", "email", "profile"}`
       - Endpoint: `endpoints.Google` (from `golang.org/x/oauth2/endpoints`)
    2. Create `GitHubConfig(clientID, clientSecret, backendURL string) *oauth2.Config`:
       - RedirectURL: `backendURL + "/auth/oauth/github/callback"`
       - Scopes: `[]string{"user:email"}`
       - Endpoint: `endpoints.GitHub`
    3. Create `FetchGoogleProfile(ctx context.Context, token *oauth2.Token, cfg *oauth2.Config) (*OAuthProfile, error)`:
       - Use `cfg.Client(ctx, token)` to get authenticated HTTP client.
       - GET `https://www.googleapis.com/oauth2/v3/userinfo`
       - Decode JSON response: `sub`, `email`, `email_verified` (bool), `name`, `picture`.
       - Return OAuthProfile with Provider="google", ProviderUserID=sub, Email, EmailVerified from response, FullName=name, AvatarURL=picture.
    4. Create `FetchGitHubProfile(ctx context.Context, token *oauth2.Token, cfg *oauth2.Config) (*OAuthProfile, error)`:
       - GET `https://api.github.com/user` -- decode: `id` (int64), `login`, `name`, `avatar_url`.
       - GET `https://api.github.com/user/emails` -- decode array of `{email, primary, verified}`.
       - Find entry where `primary=true AND verified=true`. If not found, return OAuthProfile with EmailVerified=false and empty email (the service will reject this). CRITICAL: GitHub `/user` may return `email: null` for private-email users (Pitfall 8-G). Always use `/user/emails`.
       - Return OAuthProfile with Provider="github", ProviderUserID=strconv.FormatInt(id, 10), Email from primary verified email, EmailVerified=true (since we only use verified emails), FullName=name (fall back to login if name is empty), AvatarURL.

    **Service (service.go):**
    5. Define `UserService` interface (what the OAuth service needs from user domain):
       - `GetByEmail(ctx context.Context, email string) (*user.User, error)`
       - `GetByID(ctx context.Context, id uuid.UUID) (*user.User, error)`
       - `CreateOAuthUser(ctx context.Context, input user.CreateOAuthUserInput) (*user.User, error)`
    6. Define `WorkspaceCreator` interface (for creating personal workspace on signup):
       - `CreatePersonalWorkspace(ctx context.Context, userID uuid.UUID, fullName string) error`
       - This MUST reuse the same workspace creation logic as email/password registration (Pitfall 8-E). The handler or main.go wiring will provide the concrete implementation.
    7. Create `Service` struct with fields: oauthRepo Repository, userSvc UserService, wsCreator WorkspaceCreator.
    8. Constructor: `NewService(repo Repository, userSvc UserService, wsCreator WorkspaceCreator) *Service`
    9. Implement `FindOrCreateUser(ctx context.Context, profile OAuthProfile) (foundUser *user.User, isNewUser bool, err error)`:
       - Step 1: Check existing OAuth link: `oauthRepo.FindByProviderAndID(ctx, profile.Provider, profile.ProviderUserID)`. If found, load user by ID, return (user, false, nil).
       - Step 2: CRITICAL SECURITY CHECK: If `!profile.EmailVerified`, return `(nil, false, ErrEmailNotVerified)`. Never auto-link unverified emails (Pitfall 8-B).
       - Step 3: Check existing user by email: `userSvc.GetByEmail(ctx, profile.Email)`. If found, create OAuth link via `oauthRepo.Create(ctx, existingUser.ID(), profile)`, return (existingUser, false, nil).
       - Step 4: Create new user: call `userSvc.CreateOAuthUser(ctx, CreateOAuthUserInput{Email: profile.Email, FullName: profile.FullName})`. Then create personal workspace via `wsCreator.CreatePersonalWorkspace(ctx, newUser.ID(), profile.FullName)`. Then create OAuth link via `oauthRepo.Create`. Return (newUser, true, nil).
    10. Implement `ListAccounts(ctx context.Context, userID uuid.UUID) ([]*OAuthAccount, error)` -- delegates to repository.
    11. Implement `UnlinkAccount(ctx context.Context, userID uuid.UUID, provider string, hasPassword bool) error`:
        - Count remaining OAuth accounts: `oauthRepo.CountByUserID(ctx, userID)`.
        - If count <= 1 AND !hasPassword, return `ErrCannotUnlinkLastAuth` (Pitfall 8-F).
        - Otherwise, `oauthRepo.DeleteByProvider(ctx, userID, provider)`.
    12. Implement `GetProviderConfig(provider string, cfg *config.Config) (*oauth2.Config, error)`:
        - Switch on provider: "google" -> GoogleConfig, "github" -> GitHubConfig.
        - Unknown provider -> ErrProviderNotSupported.
        - This is a helper the handler will use for initiate/callback.

    13. Verify: `cd backend && go build ./...`
  </action>
