Persisted preference tokens (source of truth = RegionalFormatsPage option values):
- date_format ∈ "YYYY-MM-DD" | "DD/MM/YYYY" | "MM/DD/YYYY" | "DD.MM.YYYY"  (default "YYYY-MM-DD")

- decimal_separator ∈ "." | ","                                            (default ",")

Reference date logic (from RegionalFormatsPage, applied to a real Date instead of SAMPLE):
  DD/MM/YYYY → `${pad(d)}/${pad(mo)}/${y}`
  MM/DD/YYYY → `${pad(mo)}/${pad(d)}/${y}`
  DD.MM.YYYY → `${pad(d)}.${pad(mo)}.${y}`
  YYYY-MM-DD (default) → `${y}-${pad(mo)}-${pad(d)}`
Reference time logic:
  h:mm A → 12-hour with AM/PM; else `${pad(h)}:${pad(mi)}`
Reference number logic:
  group int part every 3 with thousand_separator, join frac with decimal_separator.
