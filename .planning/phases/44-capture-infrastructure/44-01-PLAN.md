**1. Create `frontend/lib/hooks/use-auto-sku.ts`:**
- Export `useAutoSKU` hook that returns `{ generateSKU }`:
  ```typescript
  import { useCallback } from "react";

  export function useAutoSKU() {
    const generateSKU = useCallback((): string => {
      const ts = Date.now().toString(36);
      const rand = Math.random().toString(36).substring(2, 6);
      return `QC-${ts}-${rand}`;
    }, []);

    return { generateSKU };
  }
  ```
- The format `QC-{base36-timestamp}-{4-random-chars}` produces ~18 chars, well within the backend's VARCHAR(50) limit.
- No server validation at capture time — collision is astronomically unlikely (36^4 combinations per millisecond). If a 422 occurs at sync time, SyncManager handles it as a non-retryable error.

**2. Create `frontend/lib/hooks/use-capture-photos.ts`:**
- Import `getDB` from `@/lib/db/offline-db`
- Import `CapturePhoto` from `@/lib/db/types`
- Export `useCapturePhotos` hook that returns:
  - `storePhoto(tempItemId: string, blob: Blob): Promise<number>` — stores a photo in quickCapturePhotos with status "pending" and capturedAt = Date.now(). Returns the auto-incremented ID.
  - `getPhotosByTempItemId(tempItemId: string): Promise<CapturePhoto[]>` — retrieves all photos for a given tempItemId using the index.
  - `deletePhotosByTempItemId(tempItemId: string): Promise<void>` — deletes all photos for a given tempItemId (cleanup after upload or discard).
  - `deletePhoto(id: number): Promise<void>` — deletes a single photo by ID.

- Implementation pattern for `storePhoto`:
  ```typescript
  async function storePhoto(tempItemId: string, blob: Blob): Promise<number> {
    const db = await getDB();
    const id = await db.add("quickCapturePhotos", {
      tempItemId,
      blob,
      capturedAt: Date.now(),
      status: "pending",
    } as CapturePhoto);
    return id;
  }
  ```

- For `getPhotosByTempItemId`, use the index:
  ```typescript
  async function getPhotosByTempItemId(tempItemId: string): Promise<CapturePhoto[]> {
    const db = await getDB();
    return db.getAllFromIndex("quickCapturePhotos", "tempItemId", tempItemId);
  }
  ```

- For `deletePhotosByTempItemId`, get all by index then delete each in a transaction:
  ```typescript
  async function deletePhotosByTempItemId(tempItemId: string): Promise<void> {
    const db = await getDB();
    const tx = db.transaction("quickCapturePhotos", "readwrite");
    const index = tx.store.index("tempItemId");
    let cursor = await index.openCursor(tempItemId);
    while (cursor) {
      await cursor.delete();
      cursor = await cursor.continue();
    }
    await tx.done;
  }
  ```

- For `deletePhoto`:
  ```typescript
  async function deletePhoto(id: number): Promise<void> {
    const db = await getDB();
    await db.delete("quickCapturePhotos", id);
  }
  ```

- Wrap all functions in `useCallback` with empty dependency arrays (they only depend on `getDB` which is module-level).
- Return all four functions from the hook.
