UI-SPEC § "Feedback Family → RetroToast" (BINDING visuals):
- Region: desktop bottom-right above the 36px Bottombar (`bottom: calc(36px + var(--sp-3))`); mobile above the 56px FAB + safe-area.
- Each toast = a small Window: `border-2 border-border-ink bevel-raised`, width `min(360px, calc(100vw - 2*var(--sp-4)))`, thin pinstriped titlebar with 16px Silkscreen label + ✕ close box, body `p-sp-2` 14px.
- Semantic titlebar by type: success=mint/DONE/✓, info=blue/INFO/●, warning=butter/WARN/⚠, danger=pink/ERROR/✕.
- Behavior: auto-dismiss default ~5s; danger/error = manual-dismiss ONLY (never auto); hover pauses timer; manual close box; optional action BevelButton (e.g. UNDO).
- A11y: container role="region" aria-label="Notifications"; info/success role="status" (polite), warning/danger role="alert" (assertive); close box aria-label="Dismiss".
- Motion: slide-up+fade 120ms; respects prefers-reduced-motion → instant.
