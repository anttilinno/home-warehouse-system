What you're comparing. Same palette, same nav structure, same fonts. Only the icon family differs.
A — Lucide: reference baseline. Crisp, modern, uniform. The thing you said is too modern.
B — Pixelarticons: 800 hand-crafted pixel icons on a strict 24×24 grid. MIT-licensed, npm-published pixelarticons, ready React components. Loaded here from jsdelivr CDN as CSS masks (real production icons, not mockups). The closest production-ready retro answer.
C — Chunky 12×12 bitmap: coarser pixel grid hand-drawn for the sketch. Reads as Atari/early-Mac toolbar. Most retro of the three but would need every icon hand-crafted — no library to draw from.
Other libraries surfaced in the search (not built into this sketch but worth knowing about):
  • Pxlkit — 226+ pixel art SVG icons across 10 themed packages, plus 40+ React components and a visual icon builder.
  • Pixel Icon Library by HackerNoon — open-source pixelated icons, larger but more general-purpose.
  • Vivcybericons — 84+ 8-bit cyberpunk icons. Smaller set, more niche styling.
  • SVGHMI.pro / HMIcons — true SCADA/industrial-control symbols (valves, pumps, tanks). Mostly paid, oriented to plant/process visualization, not warehouse inventory. Probably wrong fit despite the name match.
