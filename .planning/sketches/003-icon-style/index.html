What you're comparing. Same palette, same nav structure, same fonts. Only the icon treatment differs.
A — Glyph 32px: faithful to the terminal aesthetic. Some glyphs (`▣` block vs `◐` outline) read as different weights — a known monospace quirk that gets worse the bigger you go.
B — Padded 40px: same glyphs, but the larger beveled frame turns each icon into a discrete control. Reads more "switch panel". Glyph-weight inconsistency mostly hides behind the frame.
C — Lucide strokes: matches what `frontend2/` would import for production (`lucide-react`). Uniform stroke weight. Loses CRT character but reads the cleanest, especially at smaller sizes / mobile.
